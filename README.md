# Daily Reflection Tree: Knowledge Engineering System

## 🚀 Project Overview
A fully deterministic reflection tool designed to move employees through three psychological axes: Locus (Victim vs. Victor), Orientation (Contribution vs. Entitlement), and Radius (Self-Centrism vs. Altrocentrism).

## 🛠 Tech Stack & Architecture
- **Language:** Python 3.x
- **Architecture:** Data-driven State Machine
- **No LLM Runtime:** The system uses 100% deterministic logic to ensure zero hallucinations and complete auditability.

## 📦 How to Run
1. Ensure you have Python installed.
2. Clone this repository: `git clone [YOUR_REPO_LINK]`
3. Navigate to the agent directory: `cd agent`
4. Run the engine: `python main.py`

## 🛡 Features
- **Signal Tracking:** Real-time tallying of psychological "signals" to determine dominant traits.
- **String Interpolation:** Dynamically mirrors user language (e.g., "{A1_OPEN}") without generative AI.
- **Silent Logic Gates:** Decision nodes handle complex branching behind the scenes.
